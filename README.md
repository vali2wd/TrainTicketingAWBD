## 🛠 Tech Stack

- **Database:** SQL Server
- **Frontend:** Angular

## ⚙️ Setup Instructions
1. Clone the repository.
2. Update the `ConnectionStrings` in `appsettings.json`.
3. Open the **Package Manager Console** in Visual Studio.
4. Run `Update-Database` to apply migrations.
5. Press `F5` to run.
